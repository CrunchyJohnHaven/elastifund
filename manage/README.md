| Field | Value |
|---|---|

## Scope

This route is intentionally a stub. It documents planned operator panes without claiming that private control-plane features are publicly available.

## Naming And Layout Rules

- Keep the route entrypoint as `index.html`.
- Keep stub language explicit until authenticated surfaces are implemented.
- Keep shared styling and runtime data binding in root `/site.css` and `/site.js`.
