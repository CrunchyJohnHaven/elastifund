# Develop Route

| Field | Value |
|---|---|
| Route | `/develop/` |
| Entry file | `develop/index.html` |
| Status | `active` |
| Purpose | Contributor onboarding and safe paper-mode boot path |
| Shared assets | `/site.css`, `/site.js` |

## Scope

Use this route to explain the shortest verified contributor workflow and safe defaults before code edits.
For internal cohort onboarding and the first-run handoff, pair it with `/build/`.

## Naming And Layout Rules

- Keep the route entrypoint as `index.html`.
- Keep route-specific copy in this directory only.
- Keep shared styling and runtime data binding in root `/site.css` and `/site.js`.
