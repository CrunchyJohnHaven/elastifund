# P2-24: Advanced Trading Simulation Engine
**Tool:** CLAUDE_CODE
**Status:** READY
**Expected ARR Impact:** Confidence + optimization

## Task
Build a comprehensive simulation engine that models:
1. Realistic order book dynamics (bid-ask spread, fill probability)
2. Capital compounding over time (reinvest profits)
3. Multiple concurrent positions with correlation
4. Variable position sizing (Kelly criterion)
5. Transaction costs (maker/taker fees)
6. Time-varying edge (edge decays as market becomes efficient)
7. Regime changes (periods of high/low opportunity)
8. Monte Carlo: 10,000 paths with confidence bands
9. Output: equity curves, drawdown charts, return distributions
10. Parameter optimization: grid search over thresholds, sizes, models
