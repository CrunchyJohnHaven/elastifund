# P3-21: Advanced ML Feature Engineering
**Tool:** CLAUDE_DEEP_RESEARCH
**Status:** READY

## Prompt

```
Research advanced feature engineering for prediction market trading:

1. Can we train a lightweight ML model (logistic regression, XGBoost) on top of LLM outputs to improve calibration?
   - Features: Claude probability, confidence, question length, market category, volume, time-to-resolution
   - Target: actual outcome
   - Would this outperform raw LLM estimates?

2. Embedding-based similarity: Can we use text embeddings to find markets similar to past winners?

3. Time-series features: price momentum, volume trends, order flow imbalance

4. What ensemble methods work best for probability aggregation? (linear pools, logarithmic pools, beta-transformed aggregation)

5. Any papers on using reinforcement learning for prediction market trading?
```
