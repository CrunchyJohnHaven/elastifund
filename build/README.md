# Build Route

| Field | Value |
|---|---|
| Route | `/build/` |
| Entry file | `build/index.html` |
| Status | `active-hidden` |
| Purpose | Internal employee onboarding packet and first-run execution handoff |
| Shared assets | `/site.css`, `/site.js` |

## Scope

Use this route for the no-assumption setup path, the internal FAQ, and the first 20-minute execution loop for new contributor cohorts.

## Naming And Layout Rules

- Keep the route entrypoint as `index.html`.
- Keep the route out of the primary nav.
- Keep route-specific copy in this directory only.
- Keep shared styling and runtime data binding in root `/site.css` and `/site.js`.
