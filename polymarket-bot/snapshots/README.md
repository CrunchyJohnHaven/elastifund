# Snapshot Policy

`polymarket-bot/snapshots/` is a local historical/archive surface.

- Timestamped snapshot trees in this folder are ignored by default.
- Keep only small index/readme files in source control.
- If a snapshot must be retained as canonical evidence, store it in a dedicated archive location and reference it from docs instead of tracking the full tree here.
