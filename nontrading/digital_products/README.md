# Digital Products Surface (`nontrading/digital_products`)

This package is the research lane for productized digital-offer discovery.

## Canonical Entrypoint

```bash
python -m nontrading.digital_products --run-once --source-file <path>
```

## Responsibility Boundary

- Candidate discovery and scoring for digital product niches.
- Optional CRM sync into the non-trading revenue store.
- Elastic-ready bulk export for ranked opportunities.

This lane informs prioritization. It does not autonomously publish, list, or sell products.
