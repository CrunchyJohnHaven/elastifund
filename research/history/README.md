# Research History Index

`research/history/` stores superseded or historical research artifacts that are not canonical current-state guidance.

## Buckets

- `research/history/2026_q1_velocity_cleanup_wave1/`: first Spark velocity cleanup history pass.

## History Rules

- Move files here when one of these is true:
  - file is superseded by a canonical current-state document
  - file is explicitly integrated/completed historical context
  - file has zero live references and is not part of active operations
- Keep a short README in each history bucket with move rationale and provenance.
