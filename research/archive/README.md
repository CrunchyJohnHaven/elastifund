# Research Archive Index

`research/archive/` stores superseded or historical research artifacts that are not canonical current-state guidance.

## Buckets

- `research/archive/2026_q1_velocity_cleanup_wave1/`: first Spark velocity cleanup archive pass.

## Archive Rules

- Archive only when one of these is true:
  - file is superseded by a canonical current-state document
  - file is explicitly marked integrated/completed historical context
  - file has zero live references and is not part of active operations
- Keep a short README in each archive bucket with move rationale and provenance.
