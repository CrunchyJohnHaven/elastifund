# Live Route

| Field | Value |
|---|---|
| Route | `/live/` |
| Entry file | `live/index.html` |
| Status | `active` |
| Purpose | Public-safe runtime proof board with explicit claim boundaries |
| Shared assets | `/site.css`, `/site.js` |

## Scope

Use this route for current runtime posture, blockers, and artifact-backed status that can be shown publicly.

## Naming And Layout Rules

- Keep the route entrypoint as `index.html`.
- Keep live proof labels explicit (`live`, `blocked`, `forecast`).
- Keep shared styling and runtime data binding in root `/site.css` and `/site.js`.
