# System Adapters

Each benchmarked external system gets a subdirectory here.

## What Each Adapter Should Contain

- install and launch notes
- sanitized config templates
- adapter glue for the generic harness lifecycle
- runbook notes for failure injection and quarantine rules

## First Scheduled Cohort

- Freqtrade
- Hummingbot
- NautilusTrader

Keep these adapters reproducible. The goal is to let another operator rerun the same clean-room benchmark without folklore.
