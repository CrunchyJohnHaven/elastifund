Published benchmark evidence belongs here.

## Expected Artifacts

- Sanitized configs.
- Build logs.
- Smoke and soak summaries.
- Score breakdowns.
- Artifact manifests.

## Naming Rule

- Use run-oriented folders or files keyed by benchmark run IDs.
- Maintain a `latest` pointer artifact only when at least one completed run exists.
- Keep timestamped history immutable once published.

## Rule

This directory should stay sparse until real clean-room runs finish. No placeholder scores. No speculative rankings.
